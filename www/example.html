Homicide Report Example Usage

Publish a list of homicides:

Change the city below: El Paso, Denver, Nashville, Louisville, Portland

Note: El Paso and Denver are the only cities with homicide data.

Markup used to publish this list:

Note: Change the "city" value to your city. Replace spaces in the city name with hyphens.

    <script src="http://extras.denverpost.com/app/homicide-report/application.js"></script>

    <iframe src="http://extras.denverpost.com/app/homicide-report/list.html?city=el-paso" frameborder="0" allowfullscreen scrolling="no" seamless></iframe>

    <style type="text/css">

    iframe.homicides { width: 100%; height: 400px; }

    </style>
